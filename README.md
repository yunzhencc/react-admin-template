## ⌨️ 本地开发

请使用 [pnpm](https://pnpm.io/zh) 作为包管理工具

```sh
# 克隆项目
git clone git@github.com:yunzhencc/react-admin-template.git

# 进入项目目录
cd react-admin-template

# 安装依赖
pnpm install

# 启动开发服务器
pnpm start
```

## 安装依赖

### 应用依赖

切换到需要安装依赖的应用目录下，执行以下命令安装依赖：

```sh
pnpm add antd --save-catalog-name core
```

### 开发依赖

在项目根目录下，执行以下命令安装依赖：

```sh
pnpm add vite --save-catalog-name dev
```
